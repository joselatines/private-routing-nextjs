# Private routing - Nextjs 💻

## Description 📚
 A simple project built in Next js that contains privates routes where only allowed users can access to the dashboard 
## Installation 💡 
 First, run the development server:

```bash
npm run dev
# or
yarn dev
``` 
## App features 🪐
 - Log In
- Log Out
